# Delivery Roadmap (Draft)

## Phase 0 — Discovery (Week 0-1)

- Validate requirements with stakeholders.
- Audit Wix <> Lightcast flows using Screaming Frog data.
- Confirm legal/compliance needs for data capture.

## Phase 1 — Foundation (Week 2-4)

- Scaffold Next.js PWA + Supabase + Stripe.
- Implement auth, profile store, and code generation pipeline.
- Build MVP checkout and redemption experience.
- Wire transactional emails (purchase confirmation, access instructions).

## Phase 2 — Automation & Analytics (Week 5-6)

- Layer in lifecycle journeys (welcome, reminder, churn).
- Expose admin dashboard/reporting.
- Import historical Lightcast customers (if provided).
- QA across desktop/mobile; implement observability.

## Phase 3 — Launch (Week 7)

- Content freeze; final data migration.
- Performance hardening + accessibility sweep.
- Enable CNAME routing + smoke test production.
- Train client team; document runbooks.

## Phase 4 — Optimization (Post-launch)

- Experiment with bundles/subscriptions.
- Add referral or affiliate tracking.
- Expand analytics warehouse + marketing integrations.

Timeline subject to change once scope is finalized; update with actual dates when scheduled.
